# SisanjeHR

Algebra University College - Semester VI - Designing and Developing a Complete Application Solution

---

### **Overview**
SisanjeHR is a system that consists of two applications, a Windows desktop application and a web application.
Windows desktop application is for owners and employees of hair salons to manage hair salons and customers.
Web application is for customers to find hair salons and book appointments in one place.

---

### **Tech Stack**
- WPF App .NET Framework 4.8
- ASP .NET MVC Web App .NET Framework 4.8
- ASP .NET MVC Web API .NET Framework 4.8
- Microsoft SQL Server
- Entity Framework 6
- XAML & HTML - CSS - JS
