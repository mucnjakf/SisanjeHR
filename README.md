# Sisanje.hr

Algebra University College - Semester VI - Designing and Developing a Complete Application Solution

---

### **Overview**
The idea of Sisanje.hr project is to create a system that will serve web application to users as a quick and intuitive find of hair salons in one place, and desktop application that will serve owner and employees of the hair salon to easily and efficiently promote their services.

---

### **Tech Stack:**
- WPF App .NET Framework
- ASP .NET MVC Web App .NET Framework
- ASP .NET MVC Web API .NET Framework
- Microsoft SQL Server
- Entity Framework
- XAML & HTML - CSS - JS
